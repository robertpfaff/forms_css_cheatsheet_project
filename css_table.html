Glossary HTML Form Tags
Tag Name Description
<form> Form Creates an interactive form with input fields to collect data from user
<label> Label Required for accessibility.All input fields (e.g., username) need label tag
<input> Input Blank field for user inputs. Also requires label tag.
<button> Button Creates clickable buttons on forms for users to submit data.


CSS Table Cheatsheet: Forms
Glossary HTML Form Attributes
Attribute Name Description
"action"" Action Form attribute that designates page link or function triggered by submit button
"target" Target Specifies where to display response received after submitting form. Options: blank (new), parent, top and self (current) window.
"method" Method Specifies the HTTP method to be used when submitting form data. Options: GET or POST
"autocomplete" Autocomplete If set to 'on', it aids in form completion by automatically filling in field values based on previously entered user values.
"novalidate"; Novalidate Disables validation of form-data (input) on sumbit.
"enctype" Enctype For POST method only. Specifies how to encode form-data when submitted to server.
